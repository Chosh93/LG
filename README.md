## 개발
### 프로젝트 받은 후 초기 작업
_**npm & yarn 사용 금지 !!!**_
```bash
    npm i -g pnpm
    pnpm install
```

### 로컬 환경 실행 (http://localhost:3000)
```bash
    pnpm run dev
```
### 로컬 환경 (https) (https://localhost:3000)
카메라 등 브라우저 API 권한 획득을 위해 사용 
```bash
    pnpm run dev:https
```

## 테스팅 (단위 테스트)
```bash
    pnpm run test
```
